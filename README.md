## Hướng dẫn sử dụng

Clone project này về máy tính của bạn và mở file **index.html**, chương trình sẽ xuất hiện trên nền web.

Nhấn nút **Run AI** để bật/tắt chế độ AI.

Nhấn nút **Reset** để đặt bàn cờ về trạng thái ban đầu.
